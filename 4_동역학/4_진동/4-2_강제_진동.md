# 강제 진동 (Forced Vibration)

## 1자유도 조화가진
- m x¨ + c x˙ + k x = F_0 cos(ω t)
- 정상상태 응답 진폭: |X| = (F_0/k)/√[(1−r²)² + (2ζ r)²],  r = ω/ω_n
- 공진 근방에서 감쇠가 응답을 제한.

## 전달률(기초가진)
- T_r = √[1 + (2ζ r)²] / √[(1−r²)² + (2ζ r)²]
- 방진설계: r > √2 영역에서 감쇠 적당히 작게, 고유수 낮추기

## 비선형/충격가진 개요
- 슈팅법/하만법/FFT 기반 분석, 시간응답 시뮬레이션 필요
